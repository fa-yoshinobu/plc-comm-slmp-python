# SLMP Connect Python

[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Modern Python library for Mitsubishi PLC communication.

## Features
- **Synchronous & Asynchronous**: Flexible API for any workload.
- **Auto-Discovery**: Simplified connection logic.
- **Type Safe**: Robust type hinting for modern IDEs.
