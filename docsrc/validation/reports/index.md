# Validation Reports

This section collects hardware evidence, tooling notes, and project-level QA
summaries.

Start here:

- [PLC Compatibility Matrix](PLC_COMPATIBILITY.md)
- [Stability and Verification Report](STABILITY.md)
- [Compatibility Policy Auto-Detection](COMPATIBILITY_POLICY_AUTO_DETECTION_2026-03-19.md)

Focused report groups:

- connection and other-station verification reports
- mixed block write behavior reports
- remote control and password validation reports

Most pages in this folder are point-in-time evidence reports. Use the matrix and
stability pages first, then drill into the dated reports only when you need the
underlying validation details.
